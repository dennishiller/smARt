Detailed Description

A custom handler that implements the ITrackableEventHandler interface.

Changes made to this file could be overwritten when upgrading the Vuforia version. When implementing custom event handler behavior, consider inheriting from this class instead.

event tracking handler only for video-feature

Since we have different image targets to track and video-feature has a specific behaviour when it is tracked, we created an extra tracking event handler script. It is a given script from Vuforia with small extended functions for the video-feature.
